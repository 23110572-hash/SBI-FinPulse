# DPDP Act 2023 — Data Retention (Summary)

## Storage Limitation
- Personal data must be **erased once the purpose is served** and retention is no longer necessary for legal/business reasons.
- Data Fiduciaries should not keep personal data indefinitely.

## Legal Retention
- Where other laws (e.g., RBI/PMLA record-keeping) require retention, data may be kept for that statutory period.

## Accuracy
- Data used for decisions affecting the customer must be complete, accurate and consistent.

## Security Safeguards
- Reasonable security safeguards must protect personal data against breach (encryption, access control).
- In case of a personal data breach, the Data Fiduciary must notify the Data Protection Board and affected Data Principals.

## Erasure on Withdrawal
- On withdrawal of consent or completion of purpose, data should be deleted unless legally required to retain.
